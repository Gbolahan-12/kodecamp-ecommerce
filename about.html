About HorlaSwift

HorlaSwift is a demo marketplace built to showcase frontend development and API integration. Our mission is to bring users a clean and responsive shopping experience while demonstrating modern web development practices.

This project uses a fake API for products, categories, and checkout to simulate a real-world e-commerce application.
